美国为什么这么强大?
转载
杂谈
发布日期:   2019-05-28
文章字数:   922
阅读时长:   3 分

一个政府的员工如果都是临时工，证明这个政府只是临时政府。一个国家对临时工的态度，不仅仅可以反映这个国家的体制是否先 进，还可以从深处反映一个国家的良心。

在美国肯斯尼州军用机场，曾经有一位修理工，他叫伊迪尔特，毕业于肯斯尼州工业学院，是笔者的朋友，他曾经给我讲过这么一个自己亲身经历的故事。

那年，伊迪尔特刚从学院毕业，在肯斯尼州军用机场找到了一份清洗直升机内部的工作，由于工作经验不足够，他仅被聘用为临时工，然而，他刚刚上任没多久，就似乎摊上了大事。

一天，伊迪尔特由于早上耽误了太多时间，忙碌之中把早餐都带进了工作地点。而且这天的工作量特别重，他不得不一边工作一边 吃早餐。他娴熟地拆开直升机后盖，将处于后部的SM69式升直器零件取出，一只手卸下螺丝钉，准备用小刷清除内部的灰尘，另一只手还端着一杯咖啡在喝，突然，吹过一阵寒风，伊迪尔特打了一个大喷嚏，一整杯咖啡都洒在直升机的升直器零件上，霎时间就 冒出了黑烟。伊迪尔特大吃一惊，早知道，一个高端的直升机所有的零件都是特别制作的，一旦坏了一点，整个飞机就算报废了！ 一架精密的直升机造价大约1.8亿美金，这样的负担，他承受不起。

伊迪尔特慌慌张张，不知所措，但他还是将这件事告诉了直属上级克林斯曼准将。本以为克林斯曼准将立刻会大发雷霆，谁知，他只是吃了一惊，并陪同他去检查情况。

一路上，伊迪尔特紧张地问克林斯曼准将：“我的工作是不是丢了？是不是要赔很多钱？”克林斯曼准将却回答说：“不，你只不过就是个临时工，是不必承担责任的！”

果然，事后军方官员经过商讨会议，确认了这次事故的责任，将责任完全归咎于机场官方的疏忽。因为：第一，机场没有对临时工 进行专业的清洗培训。第二，机场对临时工的强制要求促使临时工发生错误。第三，机场没有不允许工人边饮食边工作的规定。第四，机场对直升机的零件缺乏保护措施。第五，综合以上，机场的这些做法是对工人安全的不负责任，临时工不应该为机场负责， 而机场应该对临时工负责。因此，会议结果是机场赔偿伊迪尔特事故预期伤害款10000美元，并免费对其进行一年的专业培训。

临时工犯了错误不用承担责任，这是美国多数政府部门和企业所存在的规定，因为他们认为，临时工得到的工资与其应承担的责任 成正比，一旦出了事故，也是集体的“不关照”造成的，不应该由临时工担责。在这样的集体意识和民主思想，是造就美国五百强企业数量全球第一的原因，也是美国经济强大的重要原因。
