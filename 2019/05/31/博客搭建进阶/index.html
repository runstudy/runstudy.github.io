让每个新建的文章都自动添加 Front-matter

每创建一篇文章都要在上面去加标签等字段，如果后期功能多的话，可能会要加很多字段，比较麻烦，这里有个比较相对容易的方式不用每次都去添加那些字段 首先打开文件 ~/blog/scaffolds/post.md ,进行如下配置：

---
title: {{ title }}
date: {{ date }}
author: {{ author }}
tags: {{ tags }}
categories: {{ categories }}
img: {{ img }}
top: {{ top }}
password: {{ password }}
summary: {{ summary }}
---

之后再创建文章时，文章上方会自动添加这些字段
